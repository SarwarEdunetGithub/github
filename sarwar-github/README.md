# Sarwar GitHub Project

## Overview
This project is a simple website for Sharnbasva University, showcasing information about the university and providing contact options for users.

## Files
- **index.html**: The main page of the website, featuring a welcome message and navigation links.
- **about.html**: A page dedicated to providing information about Sharnbasva University.
- **contact.html**: A page that includes contact information and a form for users to reach out.
- **css/styles.css**: The stylesheet for the website, containing styles for all pages, including `contact.html`.

## Usage
To view the website, open `index.html` in a web browser. Use the navigation links to access the About and Contact pages.
